
# Campus Energy Consumption Analytics

## Overview

Campus Energy Consumption Analytics is a data analytics project that examines electricity usage patterns across various buildings within an educational institution. The project aims to identify energy consumption trends, peak usage periods, high-energy-consuming facilities, and opportunities for improving energy efficiency.

By analyzing electricity consumption data, occupancy levels, renewable energy contributions, and energy costs, the project provides actionable insights that can help institutions reduce operational expenses and promote sustainable energy management practices.


## Objectives

- Analyze electricity consumption across different campus buildings.
- Identify buildings with the highest energy usage.
- Determine peak electricity consumption periods.
- Analyze energy usage trends by day and time.
- Evaluate renewable energy contributions.
- Detect potential energy wastage and inefficiencies.
- Provide recommendations to improve energy efficiency and reduce costs.
## Dataset Information

The dataset contains electricity consumption records.

## Features


| Column Name | Description |
| :--- | :--- |
| Date | Date of energy consumption |
| Day | Day of the week |
| Building_Name | Name of the campus building |
| Building_Type | Category of building |
| Time_Slot | Morning, Afternoon, Evening, Night |
| Occupancy_Count | Number of occupants present |
| Electricity_Consumed_kWh | Total electricity consumed |
| Peak_Demand_kW | Peak power demand |
| Temperature_C | Ambient temperature |
| Renewable_Energy_kWh | Renewable energy generated |
| Grid_Energy_kWh | Energy drawn from the grid |
| Cost_Per_kWh | Electricity tariff |
| Total_Cost_INR | Total energy cost |
| Carbon_Emission_kg | Estimated carbon emissions |
| Holiday_Flag | Indicates weekend/holiday |


## Project Questions

1. Which building consumes the most energy?
2. Which time of day has the highest usage?
3. Which days consume more electricity?
4. How can energy wastage be reduced?
